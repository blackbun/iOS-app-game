# iOS App Game - Jumpy Penguin

- programmed as a side project to learn iOS in 2016
- released to Apple App Store


- Tap to play!
- Fun, character and adventure app
- Records high score

Tech Stack:
- Swift
- CocoaTouch
- UIKit
- AVFoundation
- GameKit
- SpriteKit
- CoreMotion
- Storyboards
- Game Center
